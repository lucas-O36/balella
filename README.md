1º Slide (estrutura de dados)
oque é: As estruturas de dados definem a organização, métodos de acesso e opções de processamento para a informação manipulada pelo programa.(modo de armazinar e organizar dados do computador)
-- Algoritimo manipulam dados 
-Tipos de estruturas de dados : dividido em 4 tipos
° Homogêneas:São aquelas compostos por vários elementos do mesmo tipo
° heterogêneas:São aquelas que são compostas por elementos de tipos variados dentro de si
° estática:São aquelas que possuem tamanho fixo, definido no momento de sua criação/declaração
° dinâmica:São aquelas estruturas que vão dinamicamente solicitando mais espaço de memória de acordo com a demanda exigida pelo sistema 
-- Definição
Dado:Um dado é uma representação simbólica de fatos, números, textos, imagens ou qualquer coisa que possa ser armazenada e processada por um computador
Informação: Informação é o resultado do processamento de dados, através do qual os dados são organizados, estruturados e contextualizados para fornecer significado e utilidade
Conhecimento: Conhecimento é o entendimento e a interpretação da informação, baseados na experiência, habilidades, julgamento e contexto do indivíduo
