-estrutura: é uma coleção de variaveis refereciando por um nome.
-estruturas são tipos de variaveis que agrupam dados geralmente desiguais.
-os itens de estrutura são chamadas membros ou campos.
-algumas linguangens de progamação chamam asa restruturas de resgistors.
-uma definição de estrutura forma um modelo que pode ser usado para criar variaveis de tipo daquela estruturas.
-geralmente todos os elementos de uma estrutura possuem alguma relação entre si.
-"struct" informação ao compilador que um modelo de estrutura está sendo definida.
todos os elementos que estiverem entre chaves são membros ou campos de estrutura 
campos são de dois tipos 
estrutura endereço
estrutura de carta
estrutura livro
ponto espaço bidimensional duas cordenadas (x e y)
